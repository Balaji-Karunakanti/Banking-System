A simple Banking System implemented in Java, demonstrating Encapsulation and Input Validation.

📌 Features 
✔ Create an account
✔ Deposit money
✔ Withdraw money (with validation)
✔ Check account balance
✔ Display all accounts

🛠 Technologies Used
Java
IntelliJ IDEA

Clone the repository:
